# 计费规则

## 计费项

目前免费，后续可能的收费项

| 计费项目     | 描述                                                         | 单位     | 收费方式 |
| ------------ | ------------------------------------------------------------ | -------- | -------- |
| 共享虚拟通道 | 共享虚拟端口开出的虚拟通道，视情况可能收费                   | 元/条/月 | 预付费   |
| 出云流量     | 一个虚拟路由器从云上到云下方向的流量，不足1MB的部分按照1MB收费 | 元/MB/天 | 后付费   |
